Simple theme for blogging.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Description ==

Theme used as an example for the course *Développer son thème WordPress*.

It implements all the basic features expected from a WordPress theme, except registering block styles and block patterns : 
 - custom logo 
 - custom background
 - custom header
 - custom menu
 - featured images (post thumbnails)
 - A single widget area in the footer.

It provides a simple grid view for blog listing page, and single column view for single posts and pages. 

There are three menu locations
 - The main menu slot features your main menu. On mobile, the menu is displayed in a dropdown overlay.
 - The topbar menu allows you to add a thin bar at the top of the site for an alternative menu.
 - The footer menu allows you to do the same after the footer.

You can add icons to menus by simply adding a CSS class to the menu item, corresponding to a slug in the form `icon-$slug`

Supported icons: 
 - icon-user
 - icon-cart
 - icon-menu
 - icon-search
 - icon-next
 - icon-previous
 - icon-video
 - icon-audio
 - icon-twitter
 - icon-facebook
 - icon-linkedin
 - icon-github
 - icon-instagram

This theme doesn't include any options page. Everything is handled in the customizer. Go to front end of your site while logged in and click 'Customize' to see the available options.

The theme supports all standard customizer options, like 'Site Identity' and 'Background Image'.

The theme offers simple support for the built-in blocks of the new WordPress editor, including a custom color palette, custom font sizes and wide and full alignments.

== Frequently Asked Questions ==

= Is it a block theme ? =

No, this is a simple example of a classic theme. Use it as an example for building your own themes.
